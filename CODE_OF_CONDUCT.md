# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), in full.

By participating — as a contributor, reviewer, or user filing issues — you agree to abide by its terms.

## Reporting

Reports of policy concerns can be sent privately to the maintainer at `taka.ariyaberg@uu.se`. Reports are confidential. Do not file public issues for these.

## Scope

The Code of Conduct applies in all project spaces — the GitHub repository (issues, PRs, discussions), email correspondence about the project, and any in-person or online events organized around it.
