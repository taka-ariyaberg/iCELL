|---|---|---|
| `plate_type/*.json` | tracked | Plate dimension definitions; selected by the plate type chosen in the web app. See [`plate_type/README.md`](plate_type/README.md). |

## Adding a new plate type

See [`plate_type/README.md`](plate_type/README.md). The filename stem (without `.json`)
